TimeMachine
======

${yyyy-MM-dd - 1second} // 根据pattern和偏移量计算, 下同:

${yyyy-MM-dd - 1minute}

${yyyy-MM-dd - 1hour}

${yyyy-MM-dd - 1day}

${yyyy-MM-dd - 1week}

${yyyy-MM-dd - 1month}

${yyyy-MM-dd - 1year}
